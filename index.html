Projects

MLB Spinal Rotation Application

This project was a Raspberry Pi powered handheld tablet that could be used to track rotation during PT for major league baseball team. For the software, OpenCV and python were used to track targets that were held by the subject. For hardware, laser cutting and electronics assembly were used to create the handheld tablet, with a PiCamera providing a live video feed. This was my first introduction to computer vision applications, as well as an introduction to Raspberry Pis and touchscreen monitors.

Pothole Robot

This project is a large scale autonomous road surface treatment device. Built entirely on a 7'x12' trailer, this robotic system scans potholes using a Time of Flight camera to generate a 3D pointcloud, and then processes the pointclouds to get size, location, and characteristics of the pothole. That data is used to generate a toolpath, which is sent to a gantry and extrusion system that precisely outputs asphalt to sufficiently fill the pothole. The system uses a Jetson Xavier, along with a Zaber gantry system, a KWS screw conveyor, and primarily Python software. I learned about large scale robotic systems design and integration.
